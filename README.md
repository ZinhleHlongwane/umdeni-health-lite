# Umdeni Health Lite

## 📌 Overview
Umdeni Health Lite is a simple web application designed to help households coordinate healthcare responsibilities, specifically identifying who is responsible for collecting medication for elderly family members.

## 🎯 Problem
In many South African households, elderly individuals depend on others to collect medication. This can lead to confusion and missed responsibilities.

## 💡 Solution
This application allows users to:
- Enter their name
- Select who is collecting medication
- Receive confirmation of responsibility

## 🧱 Current Features (MVP)
- Basic user input (name)
- Role selection (self, family member, caregiver)
- Confirmation message display
