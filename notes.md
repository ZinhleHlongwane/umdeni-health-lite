## HTML + JavaScript Basics

### HTML
- HTML creates the structure of the page
- Elements include:
  - h1 → heading
  - p → paragraph
  - input → user input
  - select → dropdown
  - button → clickable button

### ID
- Each element can have an id
- JavaScript uses id to find elements

### JavaScript
- JavaScript makes the page interactive

### Function
- A function is a block of code that runs when called

### Getting Input
- document.getElementById("id").value gets user input

### Displaying Output
- innerText changes what is shown on the page

### Flow of the app
1. User types name
2. User selects role
3. User clicks button
4. Function runs
5. Data is collected
6. Message is created
7. Message is displayed
